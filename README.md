1. Problem Statement
Describe the real-world problem your team is solving.
In the modern educational environment, students are required to manage multiple academic subjects simultaneously while also balancing assignments, projects, and examinations. One of the most common challenges faced by students is the lack of an effective and structured study plan. Many students either spend excessive time on certain subjects while neglecting others or fail to allocate sufficient time for revision. This imbalance often leads to incomplete syllabus coverage, increased stress, and poor academic performance.
Traditional methods of planning, such as manually creating timetables, are often rigid and do not adapt to individual learning styles or changing academic requirements. Additionally, students may not always know how to distribute topics efficiently over a given period, 
This project aims to develop a Personalized Study Planner that leverages Agentic AI to automatically generate a structured 7-day study plan based on the subjects provided by the user. The system ensures that topics are evenly distributed, includes regular revision sessions, and provides a realistic and manageable study schedule. By doing so, the project helps students improve productivity, maintain consistency, and reduce academic stress.
 
2. Proposed Model
Explain your Agentic AI system using Groq + LangChain.
The proposed system is a Personalized Study Planner powered by Agentic AI, implemented using Groq’s LLM and the LangChain framework. The system is designed to act as an intelligent assistant that understands user requirements and generates an optimized study schedule.
Agentic AI refers to systems that are capable of making decisions and performing tasks autonomously to achieve a specific goal. In this project, the goal of the agent is to create an effective and personalized study plan. The agent receives input from the user in the form of subjects and processes this input using a Large Language Model (LLM) to generate a structured output.
The system uses LangChain as an intermediary layer that simplifies interaction with the LLM. LangChain is responsible for constructing prompts, managing communication with the model, and ensuring that the input is properly formatted. The Groq API is used to access a high-performance LLM, which processes the prompt and generates the study plan.
The model works by embedding the user input into a carefully designed prompt that instructs the LLM to create a 7-day plan with topic distribution and revision. The LLM then uses its knowledge and reasoning capabilities to generate a coherent and structured response.
Overall, the proposed model demonstrates how modern AI technologies can be effectively used to solve real-world problems in education.
